# JSON Data Structure

Data format for AsyncStorage in FaithFinance app.

## Storage Keys & Format

### 1. Current User
**Key:** `@faithfinance_current_user`

Stores the currently active user profile.

```json
{
  "id": "12345",
  "name": "John Doe",
  "email": "john@faithfinance.com",
  "createdAt": "2025-12-03T10:30:00.000Z"
}
```

### 2. All Users
**Key:** `@faithfinance_users`

Array of all registered user profiles.

```json
[
  {
    "id": "12345",
    "name": "John Doe",
    "email": "john@faithfinance.com",
    "createdAt": "2025-12-03T10:30:00.000Z"
  },
  {
    "id": "67890",
    "name": "Jane Smith",
    "email": "jane@faithfinance.com",
    "createdAt": "2025-12-03T11:00:00.000Z"
  }
]
```

## Field Details

**ID Fields:**
- Format: 5-digit number string (e.g., "12345")
- Generated by `generateId()` from transactionUtils.ts

**Date Fields:**
- `date`: YYYY-MM-DD format (e.g., "2025-12-03")
- `createdAt`: ISO 8601 timestamp (e.g., "2025-12-03T10:30:00.000Z")

**Amount:**
- Type: number
- Positive values only

**Type:**
- Values: "income" | "spending" | "giving"

## Transaction Categories

**Income (5):** Salary, Part-time Job, Freelance, Gift, Other

**Spending (26):** Dining out, Groceries, Coffee & Snacks, Transportation, Gas, Parking, Housing, Rent, Utilities, Internet, Textbooks, School Supplies, Tuition, Health & Wellness, Gym Membership, Healthcare, Pharmacy, Entertainment, Movies & Streaming, Concerts & Events, Hobbies, Shopping, Clothing, Electronics, Personal Care, Other

**Giving (5):** Church, Ministry, Charity, Community Outreach, Other
